
# 🔐 Cybersecurity Notes (Day 1)

## 🔐 What is Cybersecurity?
Cybersecurity ka matlab hai computers, networks aur data ko hackers aur attacks se protect karna.

## 🎯 Kyu Important?
- Data theft rokne ke liye  
- Hacking attacks se bachne ke liye  
- Privacy protect karne ke liye  

## 👨‍💻 Types of Hackers

### 🟢 White Hat Hacker
- Ethical hackers  
- Security improve karte hai  
- Legal hote hai  

### 🔴 Black Hat Hacker
- Malicious hackers  
- Data chori karte hai  
- Illegal  

### ⚪ Grey Hat Hacker
- Kabhi legal, kabhi illegal  
- Without permission test karte hai  

## 🌐 IP Address Basics
IP Address = Internet pe har device ka unique address  

### Example
- `192.168.1.1`

## 🔢 Types of IP

### IPv4
- Example: `192.168.0.1`

### IPv6
- Example: `2001:0db8:85a3::8a2e`

## 🎯 Use
- Device identify karna  
- Communication ke liye use hota hai
