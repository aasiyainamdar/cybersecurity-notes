## 🧠 What I Learned

### 🟥 Offensive Security
- Think like an attacker  
- Weakness identify karne ke liye attacker ki tarah sochna  
- Real-world attacks samajhne me help hoti hai  

#### 🛠 Lab Practice
1. FakeBank application use ki  
2. Hidden URL find kiya application me  
3. Admin page pe attack kiya  

---

### 🟦 Defensive Security
- Think like a defender  
- Systems aur devices ko secure karna  
- Attacks detect aur stop karna  

👉 Defensive security focuses on:
- Detecting attacks  
- Investigating incidents  
- Responding before damage occurs  

👉 Offensive security se difference:
- Offensive → attack karta hai  
- Defensive → monitor aur protect karta hai  

#### 🛠 Lab Practice
1. Suspicious activity detect ki  
2. Attack identify kiya  
3. Attack stop kiya  

---

## 🎯 Key Takeaways

- Attacker mindset samajhna important hai  
- Defensive aur offensive dono roles important hai  
- Practical labs se real-world understanding milti hai
